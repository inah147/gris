# Cards Component

Este componente fornece cards flexíveis para navegação e exibição de informações, como documentos ou links de acesso rápido.

## Variantes

### 1. Início Card (.inicio-card)
Card horizontal usado na página inicial ou dashboards para navegação rápida.
Possui ícone, título, descrição e uma seta indicativa de ação.

### 2. Document Card (.doc-card)
Card vertical otimizado para listagem de documentos no Portal de Transparência.
Exibe ícone de documento, título e rodapé para metadados ou ações.

## Dependências
- `design-system.css` (tokens de cores e espaçamento)
